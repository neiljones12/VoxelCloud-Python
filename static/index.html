HVAC Chip API

Below are the available API's

The application only communicate with Units that pass their Mac Address and Serial number to the API.

Login API
API Endpoint: https://voxelcloud-demo-python.herokuapp.com/login

API Type: POST

Expected data to be passed in the body of the request

paramerter Description Accepted values
Customer_Number Customer Number Only numbers and standard characters, no dashes or colons
Password Password Standard format

An example of the body of the request

{ "Customer_Number": "AA1122", "Password": "password" }

Read API
API Endpoint: https://voxelcloud-demo-python.herokuapp.com/read

API Type: GET

Expected data to be passed in the body of the request

paramerter Description Accepted values
mac Mac Address Only numbers and standard characters, no dashes or colons
serial Serial number Standard format

An example of the body of the request

{ "mac" : "001122334455", "serial": "1" }

Write API
API Endpoint: https://voxelcloud-demo-python.herokuapp.com/write

API Type: PUT

Expected Inputs to be passed in the body of the request

paramerter Description Accepted values
comp_status Compressor Status 1=On 0=Off
fan_status Fan Status 1=On 0=Off
temp_alert Temperature alert 1= If Temp is above 80 degrees / 0= If Temp is less than 80 degrees.
temp Temperature Temp with 1 degree resolution
mac Mac Address Only numbers and standard characters, no dashes or colons
serial Serial number Standard format

An example of the body of the request

{ "comp_status": 1, "fan_status": 0, "temp_alert": 0, "temp": 80, "mac": "001122334455", "serial": "1" }

write_immediate API
API Endpoint: https://voxelcloud-demo-python.herokuapp.com/write_immediate

API Type: PUT

Expected Inputs to be passed in the body of the request

paramerter Description Accepted values
status_at_event_comp Compressor Status at event 1=On 0=Off
status_at_event_fan Fan Status at event 1=On 0=Off
status_after_event_comp Compressor Status after event 1=On 0=Off
status_after_event_fan Fan Status after event 1=On 0=Off
restart_chk_comp Compressor restart 1=Y 0=N (Restarted after 20 minutes)
restart_chk_fan Fan restart 1=Y 0=N (Restarted after 20 minutes)
temp Temperature Integer (in Fahrenheit)
mac Mac Address Only numbers and standard characters, no dashes or colons
serial Serial number Standard format

An example of the body of the request

{ "status_at_event_comp": "0", "status_at_event_fan": "0", "status_after_event_comp": "0", "status_after_event_fan": "0", "restart_chk_comp": "1", "restart_chk_fan": "1", "mac": "001122334455", "serial": "1" }

view_device_logs API
API Endpoint: https://voxelcloud-demo-python.herokuapp.com/view_device_logs

API Type: GET

Expected data to be passed in the body of the request

paramerter Description Accepted values
device_Id device Id Integer
from_date (Optional) Format: YYYY-MM-DD String
to_date (Optional) Format: YYYY-MM-DD String

An example of the body of the request

{ "device_Id": "1", "from_date": "2015-06-12", "to_date": "2018-06-12" }

add_device API
API Endpoint: https://voxelcloud-demo-python.herokuapp.com/add_device

API Type: POST

Expected data to be passed in the body of the request

paramerter Description Accepted values
Name Device Name String
Compressor_status Compressor Status 1=On 0=Off
Fan_status Fan Status 1=On 0=Off
Temperature Temperature (in F) Integer
Ip_Address IP Address String(example 255.255.255.255)
Serial_Number Device serial number String
Mac_Address Device mac number String
Communication_Frequency Device communication frequency 24 = Hourly, 48 = Every 30 minutes
Installation_Date Device installation date String (d/m/Y H:M:S)
Write_Frequency Device write frequency 0 No Write / 10 = seconds / or 15 / 30 / 60 Seconds power sampling / reporting
Write_Time Device write time Integer (In hours)
Reporting_Url Device reporting url String

An example of the body of the request

{ "Name": "Device 5", "Compressor_status": 0, "Fan_status": 0, "Temperature": 70, "Ip_Address": "10.0.0.1", "Serial_Number": "1005", "Mac_Address": "3DF2C9A6B15A", "Communication_Frequency": 24, "Installation_Date": "6/13/2018 01:02:03", "Write_Frequency": 10, "Write_Time": 10, "Reporting_Url": "https://voxelcloud-demo-python.herokuapp.com/" }
